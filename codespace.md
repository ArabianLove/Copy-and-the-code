## Welcome to your GitHub Codespace!

This repository is part of the **Code with GitHub Codespaces** course. In this course, you'll learn how to:

- Create a codespace and push code to your repository.
- Customize your codespace with a custom image and VS Code extensions.
- Personalize your codespace using dotfiles (like `setup.sh`).

### About this file

This file (`codespace.md`) is automatically opened when your codespace starts, as configured in `.devcontainer/devcontainer.json`. It helps you quickly orient yourself when you launch a new codespace.

### Dotfiles

Your repository includes a `setup.sh` script that runs automatically as part of the dotfiles setup when a new codespace is created. It installs the `sl` command-line tool and adds it to your PATH.
